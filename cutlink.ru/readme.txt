Инструкция по запуску:

1) Создать БД и назвать "cut_link".
2) Файл cut_link.sql импортировать в созданную БД, в нём хранится таблица с полями id, link и token.
3) Папку cutlink.ru положить на сервер или скопировать содержимое на свой локальный домен.
4) Перейти в браузере по http://cutlink.ru или по своему домену.
